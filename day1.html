HTML Interview Questions

1. Difference between div and span?

  • div is a block-level element and span is an inline element.
  • div should be used to wrap sections of a document and span to wrap small portion of text, images, etc.

    • 2. Difference between class and IDs?

  • class can be used to identify more than one HTML element and ID is only used to identify one single element in our HTML.
  • class assigned to an element has its name starts with "." followed by class name. and ID name starts with the "#" symbol followed by unique name assigned to it.
  • multiple class selector can be attached to an element. and only one ID selector can be attached to an element.
  • class can be applied to multiple elements so it could be multiple times on a single page. and ID is unique in a page and can only apply to at most one element.

    • 3. What are self closing tags?

  • A self closing tag in HTML are the type of HTML tag that need not to be closed manually by its closing tag which means there is no separate closing tag.
  • The tag in which no content is there is called a self closing tag.
  • The tag which has only the opening tag no closing tag are required. Self closing tag are also called empty tags.
  • Examples of self closing tags are hr tag, br tag, image tag, etc.

    • 4. How many types of headings are there? How to decide which to use where?

  • A HTML heading tag is used to define the headings of a page.
  • There are 6 level of headings defined by HTML. 6 heading are h1, h2, h3, h4 and h6.
  • h1 being the highest level and h6 being the least.
  • h1 is used for main heading (biggest in size). h2 is used for subheadings.
  • further sections under the subheadings then the h3 element is used.
  • h6 for the small heading (smallest one).

    • 5. Difference between block and inline?

  • Inline elements occupy only sufficient width. Whereas block level elements occupy full width irrespective of its sufficiency.
  • Inline element does not start in a new line. Whereas block level elements starts in a new line.
  • Inline element also allow other inline element to sit behind. Whereas block level element does not allow other element to sit behind.
  • Inline element does not have top or bottom margin. Whereas block level element have top and bottom margin too.
  • An example of inline element is span. Whereas an example of block level element is div.

    • 6. Difference between em and strong?

  • 1. em tag indicates emphasis and strong indicates stronger emphasis.
  • 2. em tag shows text as italics whereas strong makes it bold.
  • 3. The em element represents stress emphasis of its contents. and the strong element represents strong importance, seriousness,or urgency of for its contents.
  • 4. strong tag you put around a sentence or phrase to indicate that "this is more important than the surrounding text" and em generally used to indicate the stress of a word within a sentence.

    • 7. What are attributes?

  • HTML attributes are special words used inside the opening tag to control the elements.
  • All HTML elements can have attributes
  • Attributes provide additional information about elements.
  • Attributes are always specified in the start tag.
  • Attributes usually come in name/value pairs like: name="value"

    • 8. Difference between HTML tags and elements?

  • HTML tags are used to hold the HTML element.Whereas HTML element hold the content.
  • HTML tag starts with "<" and ends with ">" whereas whatever written within the HTML tag are HTML elements.
  • HTML tags are almost like keywords where every single tag has unique meaning. Whereas HTML elements specify the general content.
  • HTML tags are basically the opening and closing tag only without any content. Whereas HTML element includes tag that is opening and closing and inside tags there is the content and its attributes too.
  • Html tags are usually used to markup the start of the HTML and they are usually enclosed in angle brackets whereas HTML element has the started closing tag with the content.

    • 9. Explain br tags?

  • The br tag is used to create a line break in a text.
  • The br tag inserts a single line break.
  • The br tag is useful for writing addresses or poems.
  • The br tag is an empty tag which means that it has no end tag.

    • 10. What are semantic tags?

  • semantic tags introduce meaning to the web page rather than just presentation
  • semantic tags clearly describes the meaning to both browser and developer.
  • semantic tags indicate what content actually is rather than just how they are displayed by the browser.
